This piece is a massive choose your own adventure narrative project that took roughly a month to complete. It follows the journey of the titular "Journey man", in his path throughout the day, constantly being put down and stepped on.

This project was done in p5.js as well as photoshop. It has been coded so that when the image is clicked, it overlays a larger image and plays an mp3 of a man screaming. When combined, it looks like the square guy is screaming when you click him. Humorously, the code counts holding down the left mouse button as repeatedly clicking the mouse so while the image remains unaffected, the audio clip layers over itself into a terrifying result.

This piece comes from my AP art concentration in my senior year of highschool. I drew it using pigma microns, white gel pens, and watercolors, some of my favorite tools to use together. For my concentration, I wanted to combine elements of the fantastical with the mundane, so for this one I wanted to emulate the same feeling of an old medival painting of a knight killing a dragon.
